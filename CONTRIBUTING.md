# Guía de Contribución - AUT_SOC

¡Gracias por tu interés en contribuir a AUT_SOC!

## 📋 Reportar Bugs

1. Busca si ya existe en [GitHub Issues](https://github.com/EdwardSuite/AUT_SOC/issues)
2. Crea un nuevo issue con:
   - Título claro y descriptivo
   - Descripción detallada del problema
   - Pasos para reproducir
   - Comportamiento esperado vs actual

## 🚀 Pull Request Process

```bash
# 1. Fork el repositorio
# 2. Crear rama feature
git checkout -b feature/descripcion-corta

# 3. Commit cambios
git commit -m "feat: descripción clara del cambio"

# 4. Push a rama
git push origin feature/descripcion-corta

# 5. Abrir Pull Request en GitHub
```

**Formato de commit**:
- `feat:` Nueva funcionalidad
- `fix:` Corrección de bug
- `docs:` Cambios en documentación
- `style:` Cambios de formato
- `refactor:` Refactoring de código
- `test:` Adición de tests
- `chore:` Cambios de build, deps

## ✅ Estándares de Código

- Docstrings en todas las funciones públicas
- Type hints en Python
- Tests para nuevas funcionalidades
- Máximo 72 caracteres en título de commit
